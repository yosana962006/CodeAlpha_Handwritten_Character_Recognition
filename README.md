# Handwritten Character Recognition using CNN

## 📌 Project Overview
This project is developed as part of the CodeAlpha Machine Learning Internship.  
It focuses on recognizing handwritten digits (0–9) using a Convolutional Neural Network (CNN).

## 🚀 Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## 📊 Dataset
- MNIST dataset (handwritten digits)

## 🧠 Model
- Convolutional Neural Network (CNN)
- Achieved high accuracy using deep learning techniques

## 📈 Results
- Training Accuracy: 99.70%
- Validation Accuracy: 99.18%
- Test Accuracy: **99.24%**

## 📂 Files Included
- Task3_Handwritten_Character_Recognition.ipynb
- handwritten_model.keras

## 🎯 Conclusion
The model successfully predicts handwritten digits with high accuracy and demonstrates the effectiveness of CNN in image classification tasks.
